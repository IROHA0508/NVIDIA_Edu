# 2주차
1. 딥러닝 기초
- 딥러닝 기초 : 기본 딥러닝 이론(Gradient Descent 등등), MNIST와 Fashion MNIST를 이용한 신경망 생성
- 집값 예측(고전적 회귀 문제) -> 딥러닝으로 해결해보기, 이진분류, 다중분류  

2. CNN & 컴퓨터 비전
- OpenCV를 이용한 CNN 생성(Fashion MNIST 이용)
- Conv2D, Pooling의 의미, 패딩이 필요한 이유 
- 다중분류 (CIFAR 이용)
- OverFitting 회피를 위한 기법 : L1, L2 Regularization, DropOut, Batch Normalization
- Transfer Learning을 이용한 모델 생성 : 가중치는 가져오되 분류기만 새로 생성 (VGG16 모델의 가중치를 가져옴)
- CNN 이진분류 실습 : 강아지 vs 고양이 구분
- ResNet을 이용한 이진분류 (이전 강아지 vs 고양이 구분 실습과 성능 비교)  

3. 자연어 처리(NLP) & RNN
- 자연어 전처리(tokenize), 정수인코딩, One-Hot Encoding
- 워드 임베딩 : word2vec, GloVe, FastText
- TF-IDF : 단어/문장의 중요도 측정/ TF : 단어의 중요도 / IDF : 문서의 희귀 빈도
- 코사인 유사도, 문서간 거리 측정
- 희소 행렬과 밀집 행렬
- IMDB 문장을 이용해서 임베딩 -> 모델 학습
- RNN의 정의 -> 네이버 영화 리뷰를 이용한 긍부정 판별
- LSTM을 활용한 RNN 구현
- Seq2seq, Attention, Transformer
